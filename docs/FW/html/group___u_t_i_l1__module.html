Converts a 16bit signed number to a string, in a formatted way (like printf with "%0d"). Always terminates the result string. Note: count is the size of dst INCLUDING zero byte. Precondition: src, dst != NULL.

Parameters

[in]dstSizeSize of the destination buffer, in uint8_ts.
[in]valThe 16bit signed number to add
[in]fillFill character, typically ' ' (like for "%2d" or '0' (for "%02d")
[in]nofFillSize for the format (right aligned) string, e.g. '2' for "%2d"
