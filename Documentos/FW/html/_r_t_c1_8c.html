LDD_TDeviceData * RTC1_Init (LDD_TUserData *UserDataPtr, bool SoftInit)
 Initializes the device. There are two modes of initialization. 1) RTC device is NOT battery backed up. (SoftInit = FALSE) All registers (interrupts, counters, clock routing …) that influence RTC behavior are initialized. Note: This initialization mode doesn't preserve time and date settings. 2) RTC device IS battery backed up. (SoftInit = TRUE). Only registers that are not backed up are initialized (Interrupt priority …). RTC device is running, time and date a preserved, RTC interrupts (events) are preserved too. Allocates memory for the device data structure. This method can be called only once. Before the second call of Init() the Deinit() must be called first. More...
